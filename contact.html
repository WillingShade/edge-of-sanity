Quick setup (recommended: Formspree)

  1. Create a free Formspree account at formspree.io.
  2. Create a new form and copy the endpoint URL (looks like https://formspree.io/f/yourFormId).
  3. Open this file and set the hidden input #formEndpoint value to that endpoint.
  4. Formspree will validate submissions and can forward them to familyshade@gmail.com or you can view them in the Formspree dashboard.
